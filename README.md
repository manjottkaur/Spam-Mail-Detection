Spam Mail Detection
A Machine Learning-based web application that detects whether an email or SMS message is **Spam** or **Ham (Not Spam)**.

Technologies Used
Python
Flask
Scikit-learn
Pandas
NumPy
Joblib
HTML
CSS

How to Run
Clone the repository.
```bash
git clone https://github.com/m2934547-droid/Spam-Mail-Detection.git
```

Install the required libraries.
```bash
pip install -r requirements.txt
```

Train the model.
```bash
python train_model.py
```

Run the Flask application.
```bash
python app.py
```

Open your browser and visit:
```
http://127.0.0.1:5000
```
